What are the different fees to pay?
There are different fees to pay in making websites. The following are the different fees to pay:

-> Initial Fee: This is the main cost of developing a website. This is one time payment.
-> Maintenance Fee: This is the cost for maintaining the website, repairing error, glitches, and bugs. This is to be paid monthly.
-> Update / Change Fee: This is the cost for changing the appearance, functionalities of the website, etc... This is only to be paid every client's request to change or update their website.
-> Renew Fee: This is the cost for renewing domain of the website. This is to be paid yearly only.
-> Other Fee: This is the cost for other services but it is on client's request or developer's request and it is rare.

Failure to pay any of the fee will result on taken down of the website
